# Data Directory

This directory contains all data artifacts generated by the pipeline.
It is excluded from git (except this README and the sample directory).

## Structure

| Directory | Contents | Generated By |
|-----------|----------|-------------|
| `raw/` | JSON documents from crawler | `python scripts/run_crawl.py` |
| `processed/` | FAISS index + metadata | `python scripts/build_kg.py` |
| `kg/` | Knowledge graph triples (JSON) | `python scripts/build_kg.py` |
| `router_training/` | Auto-generated training data + model | `python scripts/train_router.py` |
| `graphrag_workspace/` | Microsoft GraphRAG workspace | GraphRAG indexing |
| `sample/` | 10 example documents for testing | Pre-included |

## Reproduction

```bash
# Step 1: Crawl legal documents
python scripts/run_crawl.py --max-docs 500

# Step 2: Build knowledge graph + FAISS index
python scripts/build_kg.py

# Step 3: Train the router
python scripts/train_router.py
```
